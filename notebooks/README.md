# Notebooks

This folder contains the main experiment notebooks used in this MSc dissertation project.

The notebooks here are the **user’s own working versions** and are the primary record of dataset inspection, baseline experiments, and later updates.

## Contents

- `01_liar_load.ipynb`  
  Load and inspect the LIAR dataset.

- `02_tfidf_baseline.ipynb`  
  Run and improve the LIAR TF-IDF baseline.

## Important note

External GitHub repositories were used only as **implementation references** and do not replace the notebooks in this folder.

The external reference links are listed in the project-level `README.md`.
