## 2.1 假新闻检测数据集与 LIAR

假新闻检测研究的发展与公开数据集的建立密切相关。在早期研究中，缺乏大规模、有标注的假新闻数据集是限制该领域发展的主要因素之一。

**Wang（2017）LIAR 数据集**

Wang（2017）发布了 LIAR 数据集，这是目前假新闻检测研究中使用最广泛的公开数据集之一。LIAR 包含从 PolitiFact.com 收集的 12,836 条政治陈述（statement），每条陈述由专业事实核查员标注，分为六个类别：true、mostly-true、half-true、barely-true、false、pants-fire。数据集还提供了说话者信息、语境、来源等元数据。

LIAR 的主要贡献在于：第一，提供了大规模真实政治假新闻语料；第二，提供了比传统二元分类更细粒度的可信度标签。然而，LIAR 也有明显局限：六分类体系在实际应用中较难区分，且政治领域的语言特征未必能泛化到其他领域的假新闻检测任务。

许多后续研究将 LIAR 的六类标签映射为二元标签（real / fake），以简化任务并与其他数据集进行比较。这种映射方式的合理性以及边界类别（如 half-true）的处理方式是研究中常见的挑战。

**Shu et al.（2020）FakeNewsNet**

Shu 等（2020）发布了 FakeNewsNet 数据集，这是另一个在假新闻研究中被广泛使用的公开数据集。与 LIAR 不同，FakeNewsNet 包含社会情境信息，如新闻文章的传播路径、社交媒体用户的转发行为等。数据集来源于 PolitiFact 和 GossipCop 两个不同领域（政治类和娱乐类），使其更适合跨领域研究。

FakeNewsNet 的贡献在于引入了多模态和多维度特征，将新闻内容、社会语境与时序传播信息结合。其局限在于社会情境特征的获取需要额外的数据接口，在某些研究中不易直接使用。

**Augenstein et al.（2019）MultiFC**

Augenstein 等（2019）在 EMNLP 2019 发表了 MultiFC 数据集，这是目前规模最大的自然产生的事实核查数据集之一，包含来自 26 个英文事实核查网站的陈述，由专业记者标注可信度。MultiFC 的特点是来源多样，覆盖政治、科学、娱乐等多个领域，并包含陈述对应的证据文本。

MultiFC 的贡献在于提供了大规模、多来源的跨域数据，有助于研究模型在不同领域间的泛化能力。其局限在于来自 26 个不同网站的标注标准并不统一，标签体系复杂，分类难度高。该数据集上最佳结果仅约 49.2% Macro-F1，说明这是一个极具挑战性的任务。

**小结**

现有假新闻数据集各有侧重：LIAR 以政治陈述为核心，FakeNewsNet 引入社会情境，MultiFC 强调多来源跨域覆盖。这些数据集之间的领域差异（domain gap）使得在一个数据集上训练的模型往往在另一个数据集上表现明显下降，这正是跨数据集泛化研究的动机之一。

## 2.2 传统方法与基于 Transformer 的假新闻检测

假新闻检测方法的发展经历了从传统机器学习到深度学习再到预训练语言模型的演进过程。

**传统机器学习方法**

早期研究主要依赖人工设计的语言特征和传统分类算法。Rashkin 等（2017）分析了不同可信度新闻的语言特征，发现存在明显的语言模式差异，如情感词汇、限定词、模糊表达的使用频率在假新闻中更高。这类研究表明，语言风格本身就包含关于可信度的信号。Pérez-Rosas 等（2018）进一步验证了基于语言学特征（如词法、句法、心理语言学特征）的分类器在检测假新闻方面的有效性，但同时指出这类方法对领域变化非常敏感，跨领域泛化能力弱。

这些传统方法的局限在于：依赖人工特征设计，难以捕捉语义层面的欺骗模式；且对领域分布变化较为敏感。

**BERT 与预训练语言模型**

Devlin 等（2019）提出的 BERT（Bidirectional Encoder Representations from Transformers）是 NLP 领域的重要突破。BERT 通过在大规模文本语料上进行掩码语言模型（MLM）和下一句预测（NSP）预训练，学习到了丰富的语言表征。在下游任务中，只需对 BERT 进行微调（fine-tuning），即可在多项 NLP benchmark 上达到当时最优性能。

**RoBERTa 的改进**

Liu 等（2019）在 BERT 基础上提出了 RoBERTa（Robustly Optimized BERT Pretraining Approach）。他们通过系统性实验发现 BERT 存在"训练不足"的问题，并通过增大训练数据量、去除 NSP 任务、动态掩码策略等方式对预训练过程进行了优化。RoBERTa 在 GLUE、RACE、SQuAD 等多项 benchmark 上超过了 BERT 的性能，并成为许多下游 NLP 任务的强力基线模型。

在假新闻检测中，RoBERTa 的更强语言表征能力使其相比 BERT 具有潜在优势，尤其是在处理语言边界模糊的可信度分类问题时。

**近期多数据集评估研究**

Papageorgiou 等（2025）对当前主流方法在多个数据集上进行了大规模评估，包括 LIAR、FakeNewsNet 等。该研究比较了传统模型和基于 LLM 的方法，并发现模型性能在数据集间存在显著差异，说明现有方法的跨数据集泛化能力仍有待提升。值得注意的是，该研究是多数据集评估（multi-dataset evaluation），即在多个数据集上分别训练和测试，而非严格的跨数据集迁移（cross-dataset transfer）实验。

**小结**

从传统机器学习到 BERT、RoBERTa，假新闻检测在同域（in-domain）任务上的性能持续提升。然而，这些方法在跨领域或跨数据集场景下的泛化能力依然有限，且训练数据的标注质量和标签体系差异进一步加大了跨数据集研究的难度。

## 2.3 基于 LLM 和推理的假新闻检测

近年来，大型语言模型（LLM）的发展为假新闻检测带来了新的可能性，但也带来了新的挑战。

**TELLER（Liu et al., 2024）**

Liu 等（2024）在 Findings of ACL 2024 发表了 TELLER 框架，这是一个面向"可信赖假新闻检测"的综合框架，核心目标是同时实现可解释性（explainability）、可泛化性（generalizability）和可控性（controllability）。

TELLER 采用双系统架构：认知系统（cognition system）使用人工设计的逻辑谓词（logical predicates）引导 LLM 生成结构化的逻辑原子（logic atoms）；决策系统（decision system）在这些逻辑原子基础上学习泛化的逻辑规则，并以规则驱动方式做出分类决策。

TELLER 在 LIAR 二元分类任务上报告的结果为：准确率 0.6773，Macro-F1 0.6697，这高于许多传统 BERT/RoBERTa 基线方法。然而，TELLER 的核心依赖是 OpenAI API（用于逻辑原子生成步骤），这在实际使用中引入了 API 调用成本和对外部服务的依赖，限制了其在学术环境中的可复现性。此外，逻辑谓词的设计需要领域知识的人工投入，这也是其局限之一。

**基于 LLM 的检测方法的更广泛讨论**

除 TELLER 外，多项研究探索了将 LLM 直接用于假新闻检测或事实核查。Hu 等（2024）研究了 LLM 在假新闻中的双重角色——既可能被用于生成虚假信息（bad actor），也可以辅助检测虚假信息（good advisor）。这一研究揭示了 LLM 在假新闻问题中的两面性，对于理解大型语言模型在信息生态中的影响具有重要意义。

Pan 等（2023）在 ACL 2023 提出了基于程序引导推理（program-guided reasoning）的事实核查方法，通过结构化推理步骤分解复杂的事实核查任务。这类方法强调可解释的推理链，而不仅仅依赖端到端的模型预测。

**小结**

基于 LLM 和推理的方法在性能上往往优于简单的 BERT/RoBERTa fine-tuning，但同时带来了更高的运行成本（API 费用、计算资源）和更复杂的系统依赖。这些方法在 MSc 规模的研究项目中完整复现的可行性有限，因此通常作为高级文献对比参考（advanced literature comparison）而非直接复现的基线。

## 2.4 类别不平衡与评估指标

假新闻检测数据集通常存在类别不平衡问题，这对模型训练和评估指标的选择有重要影响。

**类别不平衡问题**

Henning 等（2023）系统综述了 NLP 任务中的类别不平衡问题，总结了主要应对策略，包括数据层面的重采样方法（oversampling、undersampling）和算法层面的加权损失函数（class-weighted loss）等。研究指出，类别不平衡不仅影响训练稳定性，也会导致评估指标（如 accuracy）产生误导性结论。

在假新闻检测中，训练数据通常存在真新闻多于假新闻的不平衡现象（即使在标注平衡的数据集如 LIAR 中，二元映射后也可能出现比例偏差）。这使得模型倾向于预测多数类别（real），从而在 fake 类别上召回率（recall）偏低。

Johnson 和 Khoshgoftaar（2019）对深度学习场景下的类别不平衡方法进行了调查，发现加权损失函数在许多场景下比数据层面的重采样更为有效和稳定。

**评估指标选择**

在类别不平衡场景下，仅使用 accuracy 作为评估指标会产生误导。Macro-F1 是更公平的评估指标，因为它对所有类别给予相同权重，避免了多数类别的压倒性影响。在假新闻检测中，FAKE recall（假新闻类别的召回率）具有特别重要的实际意义，因为漏报假新闻（false negative）的代价通常高于误报（false positive）。

**小结**

类别不平衡是假新闻检测数据集中的普遍问题。合理选择评估指标（Macro-F1 优于 accuracy）和训练策略（加权损失函数）是建立可靠基线的基础。Fake recall 应作为评估模型实用价值的重要补充指标。

## 2.5 鲁棒性与跨数据集泛化

跨数据集泛化（cross-dataset generalisation）是假新闻检测领域的核心挑战之一，也是本研究的主要研究方向。

**领域偏移问题**

Wang 等（2022）综述了 NLP 任务中的鲁棒性研究，包括对领域偏移（domain shift）、对抗样本（adversarial examples）和数据集偏见（dataset bias）的分析。研究发现，在一个数据集上达到高性能的模型往往对另一个数据集上的分布变化非常敏感，性能下降显著。这一现象在假新闻检测中尤为突出，因为不同来源的假新闻在语言风格、主题领域和标注标准上差异较大。

**领域自适应预训练**

Gururangan 等（2020）提出了领域自适应预训练（Domain-Adaptive Pretraining, DAPT）方法：在特定领域的语料上对预训练语言模型进行二次预训练，使模型更好地适应目标领域的语言特征。DAPT 在多个领域特定任务上显示出显著的性能提升，并在 NLP 跨域研究中被广泛引用。

这一思路对假新闻跨域泛化研究有直接启示：通过在目标域语料上进行适当的预训练调整，可以缓解源域模型在目标域上的性能退化。

**跨域假新闻检测**

Silva 等（2021）在 AAAI 2021 提出了"Embracing Domain Differences in Fake News"方法，明确针对跨域假新闻检测场景，通过引入领域差异感知机制，提升模型在跨域设置下的泛化能力。这是跨域假新闻检测领域的代表性工作之一，说明研究界已经意识到 in-domain 训练的局限性，并开始关注跨域泛化问题。

**多数据集评估的意义与局限**

Papageorgiou 等（2025）在多个数据集上评估了不同假新闻检测方法，包括 LIAR、FakeNewsNet 等。该研究提供了跨数据集的性能比较视角，但其实验设计是在各数据集上分别训练和测试（multi-dataset evaluation），而非严格的跨数据集迁移实验（即在一个数据集上训练、在另一个数据集上直接测试）。这两者之间的区别对于评估真实泛化能力至关重要。

**小结**

现有研究表明，假新闻检测模型在跨数据集场景下普遍存在性能下降，而且这种下降往往大于在域内（in-domain）场景中的随机波动。领域自适应方法（如 DAPT）和跨域训练策略（如 Silva et al. 2021）提供了部分解决思路，但仍未完全解决跨数据集泛化问题。这一问题的根本原因在于不同数据集的标注标准、语言特征和主题领域差异。

## 2.6 研究空白

综合上述文献可以看出，假新闻检测领域已在以下方面取得进展：

1. 建立了多个公开标注数据集（LIAR、FakeNewsNet、MultiFC），覆盖不同领域和标注粒度。
2. 基于预训练语言模型（BERT、RoBERTa）的方法在同域任务上取得了显著性能提升。
3. 类别不平衡问题已有多种应对策略（加权损失、重采样），评估指标也逐渐从 accuracy 转向 Macro-F1。
4. 近年来出现了基于 LLM 和推理的更复杂检测框架（如 TELLER），在性能和可解释性上有所突破。

然而，以下问题仍未得到充分解决：

1. **跨数据集泛化能力不足**：大多数研究仍在单一数据集上进行评估，在一个数据集上训练的模型在其他数据集上的性能往往大幅下降。跨数据集泛化的系统性研究仍然较少。

2. **标签体系差异的影响**：不同数据集使用不同的标注标准和标签体系（如 LIAR 的六分类 vs FakeNewsNet 的二元标签），跨数据集比较和迁移因此面临额外挑战。

3. **复杂系统与可访问性的矛盾**：像 TELLER 这类先进框架虽然性能更高，但依赖 LLM API 和人工知识设计，在资源有限的研究环境中难以复现，限制了其实际应用价值。

4. **类别不平衡在跨域场景下的影响**：类别比例在不同数据集之间的变化可能进一步加剧跨域迁移中的性能下降，但现有研究对此关注不足。

这些空白表明，研究在 in-domain 设置下建立可靠基线之后，系统性地探索跨数据集泛化能力的研究仍具有重要价值。具体而言，探索如何在不依赖 LLM API 的情况下，通过加权训练策略和领域自适应方法，建立既准确又对分布变化具有一定鲁棒性的假新闻检测系统，是当前阶段值得研究的问题。
